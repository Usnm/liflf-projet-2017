Générateur de fichier GraphViz.

Author
Jayson KACED
Version
1

Contient la fonction ToGraph qui permet de générer un fichier GraphViz selon les caractéristiques de l'automate d'étude. Ce fichier .gv sera utilisable par la commande dot pour convertir le fichier en une image représentant le graphe avec ses transitions.
